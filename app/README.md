# Tumo-Vue

## Build Setup


```bash
# clone the project
git clone https://github.com/TyCoding/tumo-vue.git

# enter the project directory
cd tumo-vue

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
